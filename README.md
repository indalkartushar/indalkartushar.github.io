## Tushar's blogging repo

Theme credit to [Aditya Telange](https://github.com/adityatelange/hugo-PaperMod)

### PreRequisites
- Hugo >= v0.83.1
- [hugo-PaperMod](https://github.com/adityatelange/hugo-PaperMod) Theme
- GoLang > 1.16
### Writing New Posts
- `hugo new posts/your-post-name.md`


### Running Hugo Server
`hugo server`
